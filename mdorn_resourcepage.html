Four Things I Learned in Class

HTML5

is a markup language used for structuring and presenting content on the World Wide Web. It is the fifth and current version of the HTML standard.

CSS3

The way that you are able to style pages and easily style those pages without having to retype the style code into each page.

Programming Language vs Markup Languages

A Programming Language is used to write instructions that a computer can follow to perform tasks. A Markup Language is used to structure and format text, mainly used to display content.

HTML tags

Are considered the basic building blocks of any webpage
